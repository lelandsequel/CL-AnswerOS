Page Briefs are **detailed implementation guides** automatically generated for each page in your pSEO audit. They provide everything a content writer or developer needs to implement a page correctly.

```

- **Primary keyword:** SEO target keyword

```

## ğŸ’¡ How to Use Page Briefs

### For Content Writers
1. Copy the **H1** - Use as page heading
2. Copy the **Meta title** - Use in CMS/SEO plugin
3. Copy the **Meta description** - Use in CMS/SEO plugin
4. Use **Primary keyword** - Focus content around this term
5. Use **Internal links** - Add these links in your content

### For Developers
1. Use **URL Path** - Create route/page
2. Use **H1** - Add to page template
3. Use **Meta title/description** - Add to page metadata
4. Use **Primary CTA** - Add button/form
5. Use **Internal links** - Add navigation/related pages

### For SEO Teams
1. **Primary keyword** - Target this in content
2. **Intent** - Understand search intent
3. **Meta title/description** - Verify SEO optimization
4. **Internal links** - Verify linking strategy
